Audio Corpus Explorer
ABOUT

ACorEx is a tool for exploring audio corpora, similar to existing software like AudioStellar and CataRT. It was created to investigate ideas of 3D navigation and time series analysis in this space, and to help researchers and musicians to navigate through large collections of audio files. The tool provides an interface for analysing, browsing, and listening to audio files.

ACorEx was developed by Elowyn Fearne at the University of Huddersfield in England, utilising OpenFrameworks and FluCoMa-Core. The tool is open-source and free to use. It is available for Windows and MacOS.

ACorEx is still in development, for queries and suggestions, please see the contact links.
